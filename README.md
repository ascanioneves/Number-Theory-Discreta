### Instruções de compilação / execução para o ambiente linux:

   #### Compilador utilizado : G++
   ##### Instalação:
   ```bash
   sudo apt-get install g++
   ```
   ##### Compilação:
   ```bash
   g++ nome_do_arquivo.cpp -o executavel
   ```
   ##### Execução:
   ```bash
   ./executavel
   ```
